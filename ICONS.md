# Создание иконок для PWA

## Быстрый способ (онлайн генератор)

1. Перейдите на https://realfavicongenerator.net
2. Загрузите ваше изображение (минимум 512x512px)
3. Настройте параметры для разных платформ
4. Скачайте и распакуйте в папку `public/`

## Ручной способ

Создайте следующие файлы в папке `public/`:

- `favicon.ico` - 32x32px
- `apple-touch-icon.png` - 180x180px
- `pwa-192x192.png` - 192x192px
- `pwa-512x512.png` - 512x512px

## Рекомендации по дизайну

- Используйте простой, узнаваемый символ (например: 💰)
- Фон должен быть цвета `#6366f1` (фиолетовый)
- Иконка должна хорошо выглядеть в маленьком размере
- Избегайте мелких деталей

## Временное решение

Если нет времени создавать иконки, можно использовать эмодзи:
1. Перейдите на https://favicon.io/emoji-favicons/money-bag/
2. Скачайте набор иконок
3. Переименуйте и поместите в `public/`
